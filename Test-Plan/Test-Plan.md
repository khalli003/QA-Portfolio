# Test Plan

## Project

**SauceDemo**

https://www.saucedemo.com/

---

## Objective

The purpose of this testing is to verify the main functionality of the SauceDemo web application, identify defects, and ensure that critical user scenarios work correctly.

---

## Scope

The following functionality will be tested:

- Login
- Products page
- Shopping cart
- Checkout
- Logout
- Navigation
- Sorting products

---

## Out of Scope

The following functionality will not be tested:

- Performance testing
- Security testing
- Compatibility with obsolete browsers
- Database testing
