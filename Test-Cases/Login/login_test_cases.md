# Login Test Cases

## TC-001 Successful Login

Steps:

1. Open login page
2. Enter valid email
3. Enter valid password
4. Click Login

Expected Result:
User is successfully authorized.

---

## TC-002 Empty Password

Steps:

1. Open login page
2. Enter valid email
3. Leave password empty
4. Click Login

Expected Result:
Validation error is displayed.

---

## TC-003 Empty Email

Steps:

1. Open login page
2. Leave email empty
3. Enter password
4. Click Login

Expected Result:
Validation error is displayed.
